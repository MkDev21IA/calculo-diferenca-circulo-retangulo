# Problema: Um círculo de raio 2 é colocado dentro de um retângulo de lados 5 e 7
A partir desse enunciado tive a ideia de criar um programa mais amplo que solicitasse ao usuário os valores do raio do círculo, comprimento e altura do retângulo e assim calculasse essa diferença de área.

## Algumas considerações
                    -O centro do círculo deve ser o centro do retângulo
                    -Os valores do raio, comprimento e altura devem ser positivos
                    -O raio do círculo deve ter comprimento menor que a metade do comprimento do retângulo e também menor que a metade da altura do retângulo

## Como usar
Execute o arquivo `calculo_area.py` em um interpretador Python.
